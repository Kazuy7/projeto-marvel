Olá,

Este projeto foi desenvolvido com HTML, CSS, JavaScript e Bootstrap,
É uma página com estrutura simples de exibição que utiliza APIs da Marvel através do uso de array.

Na index temos um conteúdo estático com links, texto e imagens.

Já as outras três páginas seguem o mesmo padrão, estrutura deselvolvida com arrays que exibem o conteúdo das APIs em formato de CARDs.
